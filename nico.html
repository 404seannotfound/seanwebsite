📲 How to load this schedule

You have two options. Subscribing is better — if the schedule is updated here, your calendar updates automatically. Importing is a one-time snapshot.

🍏 iPhone / iPad (subscribe — recommended)

  1. Open the Settings app.
  2. Go to Calendar → Accounts → Add Account → Other.
  3. Tap Add Subscribed Calendar.
  4. Paste this URL into the Server field:
  5. Tap Next, then Save.

Or the quick way: tap the Download .ics file button above in Safari — iOS will offer to add the events to your calendar (this is a one-time import, not a subscription).

🤖 Android (Google Calendar — subscribe)

Google Calendar subscriptions have to be added from a computer, not the Android app:

  1. On a computer, open calendar.google.com.
  2. On the left, next to Other calendars, click + → From URL.
  3. Paste this URL:
  4. Click Add calendar. It will sync to the Google Calendar app on your Android within a few minutes.

One-time import instead: tap Download .ics file above, then open the file with Google Calendar to import events.

💻 Apple Calendar (Mac)

  1. Open Calendar.
  2. Menu bar: File → New Calendar Subscription…
  3. Paste this URL:
  4. Click Subscribe, then OK.

📧 Outlook (Microsoft 365 / Outlook.com)

  1. Open Outlook on the web.
  2. Go to Calendar, then Add calendar → Subscribe from web.
  3. Paste the URL above and give it a name like "Nico Pink Team".
  4. Click Import.

📬 Gmail (one-time import)

Gmail itself doesn't hold calendars — it hands them to Google Calendar. Click Download .ics file, then on a computer go to calendar.google.com → gear icon → Settings → Import & export → select the file → Import.

Note: All times are Pacific (America/Los_Angeles). When you subscribe or import, your calendar app will convert them to whatever time zone your device is set to.
